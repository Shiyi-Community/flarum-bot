# flarum-bot
一个 Flarum 机器人，基于 Python 开发。

> [!NOTE]
>
> 本 App 使用 AI 辅助开发。
>
> 感谢豆包。

这个本来是想用 ChatGPT 插件的，但是那个不支持硅基流动的 API。所以开发了如下这个 Bot 框架。

## 安装 & 启动

1. 在你的环境装好 Python。
2. 装依赖：
    ```
    pip install -r requirements.txt
    ```
3. 配置（参考配置文件，很简单的几个配置）
4. 启动：
    ```
    # 三选一
    python main.py
    python3 main.py
    py main.py
    ```
至此，机器人就可以启动成功了~
